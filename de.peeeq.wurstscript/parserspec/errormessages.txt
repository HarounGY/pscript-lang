	: PACKAGE IDENTIFIER LBRACK NL

	: RBRACK
	| PACKAGE IDENTIFIER LBRACK NL NL RBRACK NL NL RBRACK

	: PACKAGE IDENTIFIER LBRACK NL
		FUNCTION IDENTIFIER LBRACK
